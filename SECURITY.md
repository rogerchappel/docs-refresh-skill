# Security

`docs-refresh-skill` reads local project files to detect documentation drift. It should not need credentials or network access for tests, examples, or smoke checks.

## Reporting a Vulnerability

Please report security issues privately through GitHub Security Advisories for this repository when available, or open an issue without sensitive details.

## Safe Fixtures

- Do not commit private documentation, credentials, tokens, or customer data.
- Keep examples synthetic and small.
- Review generated drift reports before sharing them outside a project.
